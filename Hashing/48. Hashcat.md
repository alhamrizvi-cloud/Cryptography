# 48. Hashcat

## Overview

Hashcat is the world's fastest CPU/GPU-based password recovery tool. Supports 300+ hash types.

## Installation

```bash
# Kali / Debian
sudo apt install hashcat

# Check GPU support
hashcat -I
```

## Basic Syntax

```bash
hashcat -m <mode> -a <attack> <hashfile> <wordlist/mask>
```

## Common Hash Modes (`-m`)

| Mode  | Hash Type              |
|-------|------------------------|
| 0     | MD5                    |
| 100   | SHA-1                  |
| 1400  | SHA-256                |
| 1700  | SHA-512                |
| 1800  | sha512crypt `$6$`      |
| 500   | md5crypt `$1$`         |
| 3200  | bcrypt `$2*$`          |
| 1000  | NTLM                   |
| 3000  | LM                     |
| 5500  | NetNTLMv1              |
| 5600  | NetNTLMv2              |
| 2600  | MD5(MD5(pass))         |
| 16500 | JWT (HS256)            |

## Attack Modes (`-a`)

| Mode | Name        | Example                                          |
|------|-------------|--------------------------------------------------|
| 0    | Dictionary  | `hashcat -m 0 -a 0 hash.txt wordlist.txt`       |
| 1    | Combinator  | `hashcat -m 0 -a 1 hash.txt w1.txt w2.txt`      |
| 3    | Brute-force | `hashcat -m 0 -a 3 hash.txt ?a?a?a?a?a?a`       |
| 6    | Hybrid W+M  | `hashcat -m 0 -a 6 hash.txt words.txt ?d?d?d`   |
| 7    | Hybrid M+W  | `hashcat -m 0 -a 7 hash.txt ?d?d?d words.txt`   |

## Mask Charset Reference

| Symbol | Charset                          |
|--------|----------------------------------|
| `?l`   | abcdefghijklmnopqrstuvwxyz       |
| `?u`   | ABCDEFGHIJKLMNOPQRSTUVWXYZ       |
| `?d`   | 0123456789                       |
| `?s`   | Special characters               |
| `?a`   | `?l?u?d?s` combined              |
| `?b`   | 0x00‚Äì0xff                        |

## Useful Flags

```bash
-o output.txt          # Save cracked hashes
--show                 # Show already cracked hashes
--status               # Live status update
--force                # Ignore warnings (use in VMs)
-r rules/best64.rule   # Apply rules
--potfile-disable      # Don't use cached results (re-crack)
--username             # Input file has username:hash format
-w 3                   # Workload profile (1-4, 3=high)
```

## Common CTF Combos

```bash
# Quick MD5 dictionary
hashcat -m 0 hash.txt /usr/share/wordlists/rockyou.txt

# SHA-256 with best64 rules
hashcat -m 1400 hash.txt rockyou.txt -r rules/best64.rule

# NTLM brute up to 8 chars
hashcat -m 1000 -a 3 hash.txt ?a?a?a?a?a?a?a?a

# bcrypt with small wordlist (slow!)
hashcat -m 3200 hash.txt top10000.txt -w 1
```
