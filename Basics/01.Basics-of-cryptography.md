## 📘 Basics of Cryptography

### 🔐 What is Cryptography?

Cryptography is the practice of securing information by converting it into a form that cannot be understood by unauthorized people.

* Protects **confidentiality**, **integrity**, and **authentication**
* Used in passwords, HTTPS, banking, messaging apps, etc.

## 🧩 Key Terminology

* **Plaintext** → Original readable data
* **Ciphertext** → Encrypted (unreadable) data
* **Encryption** → Converting plaintext → ciphertext
* **Decryption** → Converting ciphertext → plaintext
* **Key** → Secret value used for encryption/decryption

## 🔑 Types of Cryptography

### 1. Symmetric Key Cryptography

* Same key used for encryption and decryption
* Fast and efficient
* Problem: Key sharing

**Examples:**

* AES (Advanced Encryption Standard)
* DES (Data Encryption Standard)

---

### 2. Asymmetric Key Cryptography

* Uses **two keys**:

  * Public key (shared)
  * Private key (secret)
* More secure, but slower

**Examples:**

* RSA
* ECC (Elliptic Curve Cryptography)
