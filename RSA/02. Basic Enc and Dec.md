# RSA Basics — Practice Examples

RSA is a public-key cryptosystem from cryptography used for secure encryption and decryption.

## Mathematical Form

```text
n = p × q
φ(n) = (p − 1)(q − 1)

Choose e such that:
gcd(e, φ(n)) = 1

d ≡ e⁻¹ mod φ(n)

Encryption:
C = M^e mod n

Decryption:
M = C^d mod n
```

## Python Equivalent

```python
n = p * q
phi = (p - 1) * (q - 1)

import math
math.gcd(e, phi)

from sympy import mod_inverse
d = mod_inverse(e, phi)

C = pow(M, e, n)
M = pow(C, d, n)
```

---

# Formula Comparison Table

| Step        | Math Formula      | Python Code           |
| ----------- | ----------------- | --------------------- |
| Modulus     | n = p × q         | `n = p * q`           |
| Totient     | φ(n) = (p−1)(q−1) | `phi = (p-1)*(q-1)`   |
| GCD Check   | gcd(e, φ(n)) = 1  | `math.gcd(e, phi)`    |
| Private Key | d ≡ e⁻¹ mod φ(n)  | `mod_inverse(e, phi)` |
| Encryption  | C = M^e mod n     | `pow(M, e, n)`        |
| Decryption  | M = C^d mod n     | `pow(C, d, n)`        |


# Example 1 — Encryption

### Given

```text
p = 3, q = 11, e = 3, M = 4
```

### Steps

```text
n = 33
φ(n) = 20

C = 4^3 mod 33 = 31
```

### Result

```text
Ciphertext = 31
```

# Example 2 — Decryption

### Given

```text
p = 5, q = 11, e = 3, C = 9
```

### Steps

```text
n = 55
φ(n) = 40

d = 27

M = 9^27 mod 55 = 4
```

### Result

```text
Decrypted Message = 4
```

# Example 3 — Key Generation

### Given

```text
p = 7, q = 17, e = 5
```

### Steps

```text
n = 119
φ(n) = 96

gcd(5, 96) = 1

d = 77
```

### Result

```text
Public Key  = (5, 119)
Private Key = (77, 119)
```

# Example 4 — Full RSA Cycle

### Given

```text
p = 11, q = 13, e = 7, M = 9
```

### Steps

```text
n = 143
φ(n) = 120

d = 103

C = 9^7 mod 143 = 48
M = 48^103 mod 143 = 9
```

### Result

```text
Ciphertext = 48
Decrypted  = 9
```

---

# Summary

```text
Encryption: C = M^e mod n
Decryption: M = C^d mod n

Public Key  = (e, n)
Private Key = (d, n)
```
