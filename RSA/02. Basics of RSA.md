
# RSA Basics — Practice Examples

# Example 1 — RSA Encryption

### Given

```text
p = 3
q = 11
e = 3
M = 4
```

## Step 1 — Compute Modulus

```text
n = p × q = 3 × 11 = 33
```

## Step 2 — Compute Totient

```text
φ(n) = (p − 1)(q − 1)
φ(n) = (3 − 1)(11 − 1) = 2 × 10 = 20
```

## Step 3 — Encryption Formula

```text
C = M^e mod n
```

## Step 4 — Compute Ciphertext

```text
C = 4^3 mod 33
C = 64 mod 33 = 31
```

## Result

```text
Ciphertext = 31
```

## Python Code

```python
p = 3

e = 3

n = p * q
C = pow(M, e, n)

print("Ciphertext:", C)
```

# Example 2 — RSA Decryption

### Given

```text
p = 5
q = 11
e = 3
C = 9
```

## Step 1 — Compute Modulus

```text
n = p × q = 5 × 11 = 55
```

## Step 2 — Compute Totient

```text
φ(n) = (p − 1)(q − 1)
φ(n) = (5 − 1)(11 − 1) = 4 × 10 = 40
```

## Step 3 — Compute Private Key

```text
d ≡ e⁻¹ mod φ(n)
d ≡ 3⁻¹ mod 40 = 27
```

## Step 4 — Decryption Formula

```text
M = C^d mod n
```

## Step 5 — Recover Message

```text
M = 9^27 mod 55 = 4
```

## Result

```text
Decrypted Message = 4
```

## Python Code

```python
from sympy import mod_inverse

p = 5
q = 11
e = 3
C = 9

n = p * q
phi = (p - 1) * (q - 1)

d = mod_inverse(e, phi)

M = pow(C, d, n)

print("Decrypted Message:", M)
```

# Summary

Encryption:

```text
C = M^e mod n
```

Decryption:

```text
M = C^d mod n
```

Private key:

```text
d ≡ e⁻¹ mod φ(n)
```
